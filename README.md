# **App terminal SV**

## 🚀 Sobre o projeto

O **App terminal SV** foi pensado desde o início para ser um projeto extremamente grande em 1 único arquivo, demonstrando que é possível manter um código grande organizado e legível quando bem estruturado.

Também haverá o **App terminal SV 2**, uma versão modular dividida em vários arquivos, facilitando o estudo, a manutenção e a criação de novas funcionalidades.
Acesse em: "quando disponível adicionaremos o link"

Tambem lançarem a versão web que tera interface (HTML & CSS) e será dividido em arquivos.
Acesse em: "quando disponível adicionaremos o link"
